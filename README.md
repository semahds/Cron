# Cron

> Приложение для управления школьным расписанием и оценками

<br>

## Возможности

### 📅 Уроки
Список уроков на день с номером кабинета и временем. Текущий урок подсвечивается, прошедшие — затемняются. Вверху отображается текущий урок, сколько осталось и какой следующий.

- Ввод вручную или загрузка фото расписания — ИИ распознаёт текст автоматически
- Индикация текущего/следующего урока с кабинетом и временем
- Сдвиг уроков вверх/вниз

### 📊 Средний балл
Калькулятор среднего балла с хранением оценок по предметам.

- Добавление предметов и оценок вручную
- Загрузка оценок из скриншота дневника через ИИ
- Цветовая индикация: 🔴 <2.5 / 🟠 <3.5 / 🟡 <4.5 / 🟢 ≥4.5
- Быстрый калькулятор без сохранения

- Настройка времени каждого урока
- Выбор темы оформления: светлая / тёмная
- Цветовые акценты: стандарт, красный, жёлтый, индиго, зелёный, голубой, фиолетовый, розовый, бирюзовый, оранжевый, лаймовый

### 👤 Аккаунт
Привязка Telegram-аккаунта для синхронизации данных и получения уведомлений.

- Синхронизация расписания и оценок через облако
- Загрузка данных с облака одной кнопкой

## Telegram-бот

После привязки аккаунта бот присылает уведомления об уроках и отвечает на команды:

| Кнопка | Что делает |
|--------|-----------|
| 📅 Расписание | Полный список уроков на день |
| 🕐 Сейчас | Текущий урок, перемена или ожидание - с временем и кабинетом |
| 📊 Оценки | Список всех предметов с оценками |

Уведомления приходят автоматически: в начале урока и в конце с анонсом следующего.
